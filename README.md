# DecodeLabs Data Analytics Internship

## Project 2: Exploratory Data Analysis (EDA)

### Objective
Analyze the cleaned e-commerce dataset to discover patterns, trends, and business insights.

### Tools Used
- Microsoft Excel

### Dataset
- E-commerce Sales Dataset

### Analysis Performed
- Basic statistics
- Product-wise analysis
- Payment method analysis
- Order status analysis
- Trend analysis
- Outlier detection

### Dashboard
The dashboard includes:
- Total Orders
- Total Revenue
- Average Quantity
- Product-wise Sales
- Payment Method Distribution
- Order Status Distribution
- Daily Sales Trend

### Final Outcome
The dataset was analyzed successfully and meaningful business insights were obtained. The dashboard helps visualize sales performance, customer behavior, and order trends.
